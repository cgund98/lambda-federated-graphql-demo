# lambda-federated-graphql-demo

## Installation

### Prequisites

- _Node_: `brew install node`
- _Pulumi_: `brew install pulumi/tap/pulumi`
- _Cosmo (wgc)_ `npm install -g wgc@latest`

### Optional Tools

- _pre-commit_: `brew install pre-commit`

### Install NPM dependencies

```bash
make install
```

### Deploy

```bash
make build
make deploy
```
